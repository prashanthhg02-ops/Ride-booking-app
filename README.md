# Ride-booking-app
Ride Booking App – A mobile or web application that allows users to book rides with nearby drivers for transportation.
Passenger – The person who books and travels in the ride.
Driver – A registered person who accepts ride requests and provides transportation.
Booking – The process of requesting a ride by entering pickup and destination locations.
Pickup Location – The place where the driver collects the passenger.
Drop Location – The destination where the passenger wants to go.
Fare – The amount charged for the ride based on distance, time, or pricing rules.
GPS Tracking – A feature that shows the real-time location of the driver and passenger.
Route Navigation – A system that guides the driver to the pickup point and destination using maps.
Payment Gateway – A secure system that processes online payments using UPI, cards, wallets, or net banking.
Ride History – A record of all completed and canceled rides.
Driver Rating – A score given by passengers based on the driver's service quality.
Passenger Rating – A score given by drivers based on the passenger's behavior during the ride.
Estimated Time of Arrival (ETA) – The expected time for the driver to reach the pickup location.
Cancellation – The option for the passenger or driver to cancel a ride before it begins.
Live Tracking – A feature that allows users to monitor the ride in real time.
Promo Code – A discount code that reduces the ride fare.
Admin Panel – A dashboard used to manage users, drivers, bookings, payments, and reports.
Ride Sharing – A feature that allows multiple passengers traveling in the same direction to share a ride.
Emergency (SOS) Button – A safety feature that lets users quickly contact emergency services or trusted contacts during a ride.
